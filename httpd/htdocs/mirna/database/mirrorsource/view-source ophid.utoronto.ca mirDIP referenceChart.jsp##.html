<b>Notes</b><br/><br/>
<span class='noteSymbol'>#</span> Targetscan Conserved uses conservation, but it is not integrated into the context score.
<br/>
<span class='noteSymbol'>§</span> PITA does not explicitly use conservation in scoring targets. However, accessible microRNA binding sites tend to show high conservation.
<br/>
